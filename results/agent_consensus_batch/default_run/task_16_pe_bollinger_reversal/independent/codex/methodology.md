# PE筛选后布林带反弹选股

实现逻辑：

1. 用创业板代码前缀 `300` 过滤股票池。
2. 从现货表读取 `市盈率-动态`，筛选 `15 <= PE <= 60`。
3. 对每只股票获取截至 `2024-08-15` 的日线数据。
4. 计算 20 日布林带：
   - 中轨 = 20 日简单移动平均
   - 上轨 = 中轨 + 2 倍 20 日标准差
   - 下轨 = 中轨 - 2 倍 20 日标准差
5. 反弹条件：
   - 最近 20 个交易日内至少一天 `收盘价 <= 下轨`
   - `2024-08-15` 当日 `收盘价 > 中轨`
6. 量能条件：
   - 最近 5 日平均成交量 `> 最近 20 日平均成交量`
7. 近 5 日涨幅：
   - `(截止日收盘 / 5个交易日前收盘 - 1) * 100`
8. 按 PE 升序排序，取前 8 名。
