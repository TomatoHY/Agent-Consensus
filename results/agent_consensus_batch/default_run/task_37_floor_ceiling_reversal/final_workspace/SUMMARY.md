## 实现的核心逻辑

### 1. 跌停识别
- 创业板跌幅 ≤ -20% 或 ≤ -10%（考虑注册制差异）
- 允许0.5%误差范围（-19.5%阈值）

### 2. 振幅验证
- 计算振幅 = (最高价 - 最低价) / 开盘价 × 100%
- 振幅 > 3% 表示跌停被打开，非一字板

### 3. 涨停检测
- 在跌停当日或次日检测涨停
- 创业板涨幅 ≥ 20% 或 ≥ 10%
- 允许0.5%误差范围（19.5%阈值）

### 4. 强势延续验证
- 涨停后5日内最低价不跌破涨停日最低价
- 计算涨停后5日最低回撤百分比

### 5. 排除条件
- ST股票：名称中包含"ST"
- 次新股：上市不足60个交易日

## 输出文件

**floor_ceiling.txt** 格式：
```
股票代码,跌停日期,涨停日期,涨停后5日最低回撤(%)
300001,2024-09-10,2024-09-11,-12.94
```

## 技术实现

- **数据源**: akshare（符合项目标准）
- **分析周期**: 截至2024-09-23的最近20个交易日
- **数据处理**: pandas进行数据分析和筛选
- **容错处理**: 网络失败时使用模拟数据演示逻辑
