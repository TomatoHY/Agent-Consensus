筛选口径:
1. 行业趋势: 医药/新能源/半导体/消费电子四类行业，按等权方式计算截至2024-12-13的近20日行业涨幅，要求 > 5%。
2. 个股超越行业: 个股近20日涨幅 > 所属行业20日涨幅 × 1.3。
3. 技术指标: 收盘价同时高于MA20和MA60；MACD histogram近5日持续为正且逐日递增；14日RSI在50-70。
4. 量能与K线质量: 近10日阳线天数 >= 6；近10日平均成交量 > 近60日平均成交量。
5. 基本面: PE > 0 且 PE < 100。
